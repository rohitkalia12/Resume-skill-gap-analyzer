# resume-skill-gap-analyzer

## Overview
Resume Skill Gap Analyzer is a web-based application that helps students and job seekers identify the gap between their current skills and the skills required for a specific job role. By uploading a resume and selecting a target role, users receive clear insights into missing skills and improvement areas.

## Problem Statement
Many candidates apply for jobs without understanding why they are not shortlisted. This project solves that problem by analyzing resumes against job roles and highlighting skill gaps, making career preparation more focused and efficient.

## Features
- Upload and analyze resumes
- Select job roles for comparison
- Identify matched and missing skills
- Get improvement suggestions
- Simple and beginner-friendly UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Git & GitHub

## How It Works
1. Upload your resume file  
2. Select a desired job role  
3. Click on analyze  
4. View matched skills and missing skill gaps  
5. Use suggestions to improve your profile  

## Use Case
This tool is useful for students, freshers, and career switchers who want clarity on what skills they need to learn to match industry requirements.
